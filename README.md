Juegos incluidos

1. Piedra, Papel o Tijera

Juego contra el ordenador. El bot elige de forma aleatoria.
El jugador puede repetir rondas hasta dejar la entrada en blanco.

2. Adivina el número

El jugador elige dificultad (1-20, 1-50 o 1-100).
Tiene 5 intentos y el juego da pistas indicando si el número es mayor o menor.
Se mide el tiempo empleado para adivinarlo.

3. Cálculo mental exprés

Se generan 8 operaciones aleatorias de suma o resta.
Hay un límite total de 35 segundos para completar todas las preguntas.
Al final se muestra la puntuación conseguida.

4. Eco invertido

El usuario escribe una frase y el programa muestra:

La frase invertida

El número de caracteres

El número de vocales
Termina cuando se introduce una línea vacía.


Cómo ejecutar el programa:

Tener Python 3 instalado.

Descargar el archivo MiniArcade.py.

Ejecutarlo desde la terminal


Extras:
Dificultad en “Adivina el número” (Fácil, Normal, Difícil)

Cronómetro en “Adivina el número” y “Cálculo mental”

Conteo de vocales en “Eco invertido”

Mensajes claros y pausas con time.sleep() para mejor experiencia
